# 快麦 ERP 官方导出地图

| 业务资源 | `--resource` | 快麦页面方向 | 稳定来源键优先级 | 时间/快照口径 |
|---|---|---|---|---|
| 三个月内订单 | `orders` | 订单管理 → 三个月内订单 | 系统订单号 | 订单创建时间 |
| 归档订单 | `orders` | 订单管理 → 归档订单 | 系统订单号 | 订单创建时间 |
| 订单商品明细 | `order_items` | 数据 → 销售主题报表 → 按订单商品明细 | 明细 ID；否则系统订单号 + 规格商家编码；同订单同规格重复行追加稳定行序号 | 创建时间 |
| 商品 | `products` | 商品管理 → 商品档案 | 系统商品 ID；主商家编码 | 当前快照 |
| SKU | `skus` | 商品/规格导出 | 系统规格 ID；商家编码；69 码 | 当前快照 |
| 当前库存 | `inventory_snapshot` | 库存管理 → 库存查询 | 仓库 + 规格 ID/商家编码 | 导出时点快照 |
| 库存流水 | `inventory_movements` | 库存管理 → 出入库/库存流水 | 单据号 + 明细/规格 ID | 出入库业务时间 |
| 供应商 | `suppliers` | 基础资料/采购 → 供应商 | 供应商 ID/编号 | 当前快照 |
| 采购单 | `purchase_orders` | 采购管理 → 采购订单 | 采购单号 | 采购创建/审核时间 |
| 售后 | `aftersales` | 售后管理 → 售后/退款单 | 售后单号 | 售后申请时间 |
| 店铺 | `shops` | 基础资料 → 店铺 | 店铺 ID/编号 | 当前快照 |
| 仓库 | `warehouses` | 基础资料 → 仓库 | 仓库 ID/编号 | 当前快照 |
| 财务 | `finance` | 财务/报表 → 账单或结算 | 结算单/账单/流水号 | 结算业务时间 |

## 字段变化处理

表格只记录方向，不证明当前租户一定拥有相同菜单或列。每个新资源先导出短范围脱敏小样并运行 preflight；真实表头未进入测试前，不执行全历史上传。

当前租户的归档订单按创建时间验证到 `2024-02-13 00:57:18`；`2020-01-01` 至 `2022-12-30` 无订单。销售主题报表必须把统计维度切换为“创建时间”，不能沿用默认“系统发货”。
