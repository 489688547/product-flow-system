## 完成标准

- 九个品牌页面可从左侧导航进入，右侧没有顶部二级导航。
- 内容主档可创建、分配、推进并持久化。
- 学习期、测试不足、重复 ID 和未对平判断符合 PRD。
- 数据中心或 NAS 不可用不阻塞生产协同。
- 既有经营、产品和平台测试不回归。
