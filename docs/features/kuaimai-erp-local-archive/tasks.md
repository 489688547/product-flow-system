# 快麦 ERP 本地原始归档与完整导入执行任务

## 执行规则

- 内联执行，不使用子代理。
- 每项先写失败测试并确认失败原因，再实现最小闭环。
- 每个任务单独验证和提交，只暂存本任务文件。
- 生产 D1、Pages、桌面归档、LaunchAgent 和 Git main 分别保留可复核证据。
- 不复制、记录或提交账号、密码、Cookie、验证码、采集令牌或个人信息。

## 任务

- [x] 本地原始归档能力
  - 文件：`scripts/kuaimai-erp-collector/archive.mjs`、CLI、归档测试和 fixture。
  - 输出：固定目录、0700 权限、稳定性门、空间门、哈希去重、APFS clone/copy、原子归档和可恢复 manifest。
  - 验证：重复文件只存一次；来源文件路径、哈希和字节保持不变。

- [x] D1 归档清单与固定范围采集令牌
  - 文件：迁移、环境清单、生成模块、runner/archive 路由、ingest 存储和 API 测试。
  - 输出：archive 幂等清单、批次关联、只存哈希的采集令牌、公司会话与采集器双认证、稳定错误码。
  - 验证：匿名、错误范围、撤销令牌、重复哈希、超大/敏感 payload、旧客户端兼容均覆盖。

- [x] 最小索引与共享业务投影
  - 文件：领域标准化、投影适配器、商品目录和货流存储、完成批次测试。
  - 输出：订单最小索引、商品/SKU、库存快照、库存流水、采购、售后、店铺、仓库和供应商标准投影。
  - 验证：完整原始字段不入 D1；只有完整批次切换快照；订单创建时间和排除“其他”口径保持一致。

- [x] 自动扫描与 LaunchAgent
  - 文件：scan/install 命令、钥匙串适配器、plist 模板、锁和契约测试。
  - 输出：每 15 分钟扫描、运行时读钥匙串、单实例、失败重试、重启恢复和无文件等待状态。
  - 验证：plist/日志不含秘密；无文件不写线上零值；重复执行幂等。

- [x] 数据中心归档状态
  - 文件：archive 查询客户端、执行记录界面和 UI 测试。
  - 输出：资源、时间范围、归档/入库状态、行数和脱敏错误；不展示绝对路径。
  - 验证：加载、空、等待、错误、无权限、成功、键盘和窄屏状态。

- [x] 现有历史文件归档与生产补数
  - 输入：桌面销售主题明细、交易订单导出及后续官方商品/库存/采购/售后文件。
  - 输出：`~/Desktop/公司数据中心/快麦ERP/` 唯一归档、处理报告、生产 archive/batch/事实和数据缺口清单。
  - 验证：源文件仍存在且哈希一致；线上批次、范围和事实数量与报告一致；缺失资源标记等待导出而不是零。

- [x] 发布与最终验收
  - 输出：环境和集成写回、全套 Definition of Done、生产迁移、Pages 部署、浏览器验收、LaunchAgent 启用、分支合入并推送 `main`。
  - 验证：本地、生产 D1、生产页面、定时任务和 Git main 五条证据链均通过；回滚步骤可执行。
