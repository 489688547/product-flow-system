# 平台连接功能设计书（底层能力）

> 当前产品入口已并入“数据中心 → 数据接入”：钉钉在“公司数据”，快麦在“ERP”。本文保留平台专属表单、安全状态和服务端契约设计；独立左侧入口与返回文案以 `docs/features/data-access-hub/design.md` 为准。

## 用户任务

老板需要快速判断哪些平台可用，并在缺少或失效时自己完成连接，不理解环境变量、密钥绑定或数据库结构。

## 信息层级

- 主信息：平台名称、Logo、当前状态、主要行动按钮。
- 辅助信息：用途、已配置字段数、最近验证时间、更新人。
- 低频信息：安全说明、回退来源、审计摘要；不显示环境变量名和加密实现。

## 页面结构

- 数据中心左侧不再单设“平台连接”，由“数据接入”目录按平台调用本能力。
- 默认页面为一个无嵌套卡片的连接清单：钉钉、快麦、阿里云各占一行，左侧为内置标识，中部为用途和状态，右侧为“配置连接”或禁用的“准备接入”。
- 点击真实已接入平台后在同页进入详情，顶部返回所属数据接入分类。不使用弹窗，避免钉钉 WebView 焦点和滚动问题。
- 详情页左侧为专属表单，右侧为连接状态、安全说明和最近操作；窄屏改为单列。

## 交互流程

1. 进入页面后并行加载全部脱敏连接状态。
2. 选择平台后，表单根据平台定义显示专属中文字段。
3. 已配置字段的输入框为空，旁边显示“已配置”；占位文案为“留空则保持原值”。
4. 任一字段被修改后启用“保存并验证”。
5. 提交期间禁用平台切换、表单和按钮，按钮显示“正在验证…”。
6. 成功后清空输入框、刷新状态并把焦点移到成功提示。
7. 失败时保留本次输入，显示安全错误和“原连接未受影响”。
8. “停用连接”使用页面内二次确认，成功后回退旧环境配置；没有回退时状态显示“未连接”。

## 组件复用

- 复用 `PageHeader`、`Button`、现有状态颜色、表单控件和焦点样式。
- 复用注册表中的平台名称、生命周期和用途，不新增第二份平台事实源。
- 不复用内部平台资料通用表单；它明确禁止凭据，职责不同。
- 不把凭据表单做成业务无关的共享 UI，因为钉钉和快麦字段、说明与验证语义不同；只共享底层布局和状态组件。

## 新增组件

### `PlatformConnectionsWorkspace`

- 职责：加载平台状态、管理清单与详情切换。
- 属性：`canManage`。
- 状态：加载、读取失败、列表、选中平台。
- 键盘：连接行使用原生按钮；Enter/Space 进入，返回按钮恢复到清单标题。

### `PlatformConnectionForm`

- 职责：按平台字段定义收集本次变更并提交。
- 属性：`definition`、`connection`、`canManage`、`onSave`、`onDisable`。
- 状态：未修改、保存中、成功、失败、无权限、已禁用。
- 复用边界：仅服务具有服务端凭据解析器和只读验证器的平台。

### `PlatformMark`

- 职责：提供稳定内置平台标识；使用品牌名称首字或已有安全资源，不从外网加载 Logo。
- 属性：`platformId`、`size`。
- 无障碍：装饰性标识 `aria-hidden`，平台名称由相邻文本提供。

## 页面状态

- 加载：三行骨架，保留布局，不显示旋转图标。
- 空数据：说明当前没有可配置的真实平台，并提供说明书入口。
- 错误：说明状态读取失败，提供“重新加载”；不把全部平台误报为未连接。
- 无权限：清单可读，详情表单禁用，显示“仅最高权限管理员可修改”。
- 禁用：尚未接入的平台按钮禁用并展示原因；停用的真实连接可以重新填写并验证启用。
- 成功：`role=status` 文案“连接已验证并启用”。
- 失败：`role=alert` 显示安全错误、“原连接未受影响”和请求编号（如有）。

## 响应式与钉钉 WebView

- 1280×800 笔记本：列表操作不换行；详情双列，主表单宽度不超过 720px。
- 768px 以下：详情单列，操作按钮可换行但保持 44px 触控高度。
- 390px：平台行改为上下结构，主操作占满可用宽度；密码输入不触发横向滚动。
- 使用 `100dvh` 现有应用壳，不创建内部固定高度滚动区。

## 交互文案

- 详情标题：对应平台名称与“平台连接”区块名
- 页面说明：维护该平台的安全连接，保存后自动验证。
- 主按钮：保存并验证
- 成功：连接已验证并启用。
- 更新失败：连接验证失败，原连接未受影响。
- 已配置字段：已配置 · 留空保持原值
- 无权限：仅最高权限管理员可以修改平台连接。
- 阿里云：准备接入 · 当前尚无可用的系统适配器

## 无障碍

- 原生 `label` 绑定输入框，密码字段支持显示/隐藏本次输入但不显示历史值。
- 状态不仅使用颜色，同时显示图标和文字。
- 焦点顺序为返回、字段、显示按钮、保存、停用。
- 错误摘要获得 `role=alert`；成功提示获得 `role=status`。
- 动效仅使用 150ms 状态过渡，遵循 `prefers-reduced-motion`。

## 视觉验收

- 1280×800：连接清单、钉钉详情、快麦详情、验证失败。
- 768×900：详情单列、按钮换行、权限只读。
- 390×844：列表、密码输入、错误和禁用平台。
- 钉钉 WebView：软键盘打开后保存按钮仍可到达，返回不丢失主导航。
