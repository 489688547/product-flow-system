# 产品说明书

## 文档类型

产品说明书

## 使用人群

公司内部管理层、产品、运营、品牌、供应链、客服和财务同事，主要在钉钉工作台内协作。

## 产品目标

让需求、产品阶段、跨部门任务、交付物和复盘在一个可执行系统中流转，并由组织架构权限控制可见与可编辑范围。

## 共享商品主数据

数据中心是 ERP 商品、SKU、主/规格商家编码和 69 码的统一导入与同步入口。产品全周期保留业务产品身份，通过目录 ID 关联 ERP 商品；供应链、供应商管理、销量和库存使用同一目录，不再各自维护一套商品编码。快麦 API 与商品档案文件是互补来源，导入和同步均为只读 ERP，不反向修改快麦。

## 产品气质

专业、克制、高效。

## 明确不做

- 不做厚重传统后台、营销页或装饰性仪表盘。
- 不使用嵌套卡片、花哨渐变和不一致的控件。

## 设计原则

1. 任务优先，权限自然表达。
2. 信息密度高但保持稳定行高和间距。
3. 组件和交互状态统一。
4. 钉钉内嵌环境优先保证可靠性。

## 无障碍与包容性

以 WCAG AA 为目标，交互不依赖颜色单独表达，保留焦点和禁用状态。
